# 🔑 CredVault
Welcome to CredVault, a sleek Flutter app integrated with Firebase Firestore for managing credentials like social media or professional accounts. Master CRUD operations (Create, Read, Update, Delete) with real-time sync and a responsive UI. Perfect for learning Firebase integration and building secure, user-friendly apps! 🚀

# ✨ Features

1. 🔥 Firebase Firestore: Real-time database for seamless credential storage.
2. 🛠️ Full CRUD Operations: Create, read, update, and delete credentials effortlessly.
3. 🔄 Real-Time Sync: Instant updates across devices.
4. 📱 Responsive UI: Smooth, adaptive design for mobile and web.
5. 🔒 Secure Management: Safely store and manage account credentials.



# 🛠️ Tech Stack

1. Flutter & Dart: Cross-platform UI framework.
2. Firebase Firestore: Real-time NoSQL database for credential storage.
3. Responsive Design: Optimized for mobile and web platforms.
